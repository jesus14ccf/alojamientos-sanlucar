# 🏡 Alojamientos Sanlúcar - Plataforma de Alquiler Vacacional

Este proyecto es una plataforma web completa de gestión de alquileres vacacionales desarrollada como **Trabajo de Fin de Grado (TFG)**. Está diseñada para mostrar las propiedades de "Casa Currito" en Sanlúcar de Barrameda, permitiendo a los clientes realizar reservas y a los propietarios gestionar su negocio de forma independiente.

## 🚀 Tecnologías Utilizadas

*   **Frontend:** Angular, TypeScript, HTML5, SCSS (Diseño 100% Responsive).
*   **Backend:** PHP (API RESTful pura).
*   **Base de Datos:** MySQL.
*   **Otros:** Optimización de imágenes (WebP/Squoosh), control de enrutamiento y protección de rutas.

## ✨ Características Principales

### Para los Clientes (Parte Pública)
*   **Catálogo de Alojamientos:** Visualización de casas con detalles, precios y galería de imágenes.
*   **Formulario de Reserva:** Sistema intuitivo para solicitar fechas de entrada y salida calculando la disponibilidad.
*   **Información Local:** Sección "Descubre Sanlúcar" y página de "Sobre Nosotros" con vídeo de presentación.
*   **Contacto:** Formulario de contacto directo y mapa de ubicación interactivo.

### Para la Administración (Panel Privado)
*   **Autenticación Segura:** Acceso restringido mediante Login.
*   **Gestión de Propiedades (CRUD):** Añadir, editar, ocultar y eliminar alojamientos.
*   **Gestor de Archivos:** Subida de imágenes al servidor para la galería de cada casa.
*   **Control de Calendario:** Bloqueo manual de fechas para evitar reservas duplicadas en otras plataformas.
